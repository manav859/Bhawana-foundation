# Bhawna Media and SEO

Use this skill for content discoverability and media handling.

## Scope

- page metadata
- social sharing metadata
- slugs and canonical URLs
- image usage and alt text
- gallery and blog presentation
- sitemap and robots preparation

## Workflow

1. Ensure each public detail page has a slug and SEO-ready metadata shape.
2. Use descriptive, non-spammy titles and meta descriptions.
3. Load large media responsibly with lazy loading where appropriate.
4. Give every meaningful image alt text tied to the surrounding content.
5. Keep gallery and blog cards consistent with the design system.

## Rules

- Use real copy and metadata, not lorem ipsum.
- Do not hide broken media behind silent failures; provide graceful fallbacks.
- Keep URLs stable once exposed.
- Prefer server-provided SEO fields with sensible defaults from global settings.
