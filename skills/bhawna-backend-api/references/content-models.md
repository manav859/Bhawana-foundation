# Suggested Content Domains

## Public content collections

- `settings`
  - organization name
  - logo
  - contact details
  - social links
  - footer content
  - SEO defaults

- `programs`
  - title
  - slug
  - excerpt
  - body
  - cover image
  - gallery
  - impact stats
  - status
  - seo

- `projects`
  - title
  - slug
  - category
  - summary
  - body
  - cover image
  - stats
  - status
  - seo

- `posts`
  - title
  - slug
  - excerpt
  - body
  - cover image
  - publishedAt
  - status
  - seo

- `events`
  - title
  - slug
  - short description
  - long description
  - start and end dates
  - venue
  - hero image
  - registration state
  - status

- `galleryItems`
  - title
  - album or category
  - image
  - caption
  - featured flag
  - publishedAt

- `testimonials`
  - name
  - role
  - city
  - quote
  - avatar or initials
  - featured flag
  - status

- `partners`
  - name
  - logo
  - url
  - sort order
  - status

## Submission collections

- `contactSubmissions`
- `newsletterSubscribers`
- `volunteerApplications`
- `donationIntents`
- `eventRegistrations`

Each submission collection should capture status, timestamps, and the minimum review metadata needed for future admin handling.
